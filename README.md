# java-cli-maven-spring-failsafe-junit-test-car-data

## Description
A POC for sprimg app using JUnit
framework with failsafe plugin.

## Tech stack
- java
- maven
  - spring
  - junit
  - failsafe

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
